# Real invoice corpus

Only invoices obtained from real inbound business traffic belong here. Do not
generate, redraw, or sanitize a demo PDF into this directory and call it real.

Before committing an invoice, confirm that the organization is authorized to
retain it in this repository and remove or encrypt regulated personal data as
required by its data-handling policy. Record the source vendor, acquisition
date, format characteristics, and the corresponding Writ `document_id` in the
week-one run report outside the invoice itself.

The directory is intentionally empty until an authorized real invoice is
available. The poisoned and forwarded variants must be derived from one of
those authorized invoices; the existing generated demo fixtures are not valid
inputs for this corpus.
